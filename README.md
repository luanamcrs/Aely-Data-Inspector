# Aely Data Inspector

Ferramenta inteligente para análise de qualidade de dados utilizando Python, Pandas e Inteligência Artificial.

## Sobre o projeto

O Aely Data Inspector foi desenvolvido para automatizar uma etapa fundamental em projetos de dados: a identificação de problemas de qualidade antes da análise ou tomada de decisão.

A aplicação permite carregar bases CSV ou Excel, analisar inconsistências, visualizar indicadores de qualidade e gerar diagnósticos executivos.

---

## Funcionalidades

- Upload de arquivos CSV e Excel
- Análise automática de qualidade dos dados
- Identificação de:
  - valores ausentes
  - registros duplicados
  - inconsistências textuais
- Score de qualidade da base
- Visualizações exploratórias
- Exportação dos problemas encontrados
- Geração de relatório executivo com IA
- Exportação de relatório PDF
